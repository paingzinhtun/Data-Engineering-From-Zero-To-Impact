# 18 - Templates

## What This Section Covers

This section contains reusable templates for project READMEs, architecture notes, pipeline documentation, data quality plans, local infrastructure, and portfolio writeups.

## Why It Matters In Data Engineering

Templates help learners practice professional documentation habits from the beginning. They also make projects easier to review, maintain, and explain in interviews or team settings.

## What You Should Be Able To Do

After using this section, you should be able to:

- Start a project with a clear documentation structure.
- Describe datasets consistently.
- Record assumptions and validation checks.
- Write operational notes for pipelines.
- Prepare portfolio case studies with business context.

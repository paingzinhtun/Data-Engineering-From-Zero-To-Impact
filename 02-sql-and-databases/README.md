# 02 - SQL And Databases

## What This Section Covers

This section covers SQL, relational databases, query design, table relationships, constraints, joins, aggregations, window functions, indexes, and database fundamentals.

## Why It Matters In Data Engineering

SQL is one of the most important skills in Data Engineering. It is used for exploration, validation, transformation, modeling, debugging, analytics, and interview assessments. Strong SQL skill makes every later stage easier.

## What You Should Be Able To Do

After studying this section, you should be able to:

- Write clear SQL queries for real business questions.
- Join related tables correctly.
- Use aggregation and window functions.
- Design a small relational schema.
- Validate data using SQL checks.
- Explain primary keys, foreign keys, indexes, and constraints.

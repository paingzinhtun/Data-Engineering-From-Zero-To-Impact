# Resources

## What This Section Covers

This section will collect supporting resources for learners, such as reading lists, glossary entries, tool references, datasets, diagrams, external links, and recommended practice material.

## Why It Matters In Data Engineering

Learners need references, but resources should support the roadmap instead of replacing it. This section should help learners go deeper while keeping the main repository focused and navigable.

## What You Should Be Able To Do

After using this section, you should be able to:

- Find supporting material for roadmap topics.
- Review common terminology.
- Locate useful datasets or practice sources.
- Compare tools at a high level.
- Continue learning without losing the practical path.

## Related Projects In This Repo

## Resource Index

- `books.md`: durable reading topics and how to choose books.
- `courses.md`: course selection criteria and stage-based course ideas.
- `official-docs.md`: stable official documentation references.
- `tools.md`: tool categories and the problems each tool solves.
- `datasets.md`: public dataset search ideas and synthetic dataset designs.
- `communities.md`: communities, etiquette, and learning habits.

## Principle

Resources should support practice. Do not collect links passively. Use each resource to build, improve, debug, or explain a project.

# 17 - Projects

## What This Section Covers

This section contains hands-on project labs that help learners apply concepts from the roadmap. Projects are practical, staged, and connected to realistic business problems.

## Why It Matters In Data Engineering

Projects are the strongest proof of learning. A good Data Engineering project shows that the learner can understand a business problem, design a data flow, write reliable code or SQL, validate outputs, document assumptions, and explain impact.

## What You Should Be Able To Do

After working through projects in this section, you should be able to:

- Build end-to-end data workflows.
- Explain source data, architecture, transformations, and outputs.
- Add data quality checks.
- Document setup and usage.
- Improve a project over multiple stages.
- Present the project as portfolio evidence.

## Related Projects In This Repo

- `beginner/`: first hands-on projects for CSV loading, API extraction, SQL analysis, and a small retail sales pipeline.
- `intermediate/`: portfolio-ready specifications for API ETL, retail warehousing, reorder analytics, dbt transformations, and Airflow orchestration.
- `advanced/`: architecture-focused projects for lakehouse design, PySpark batch processing, Kafka streaming, cloud warehousing, and AI-ready data platforms.
- `capstone/`: complete professional portfolio blueprints for retail intelligence, inventory optimization, SME platforms, and AI analytics assistants.
- Future project themes include customer 360, supplier performance, operations KPIs, data products, and industry-specific analytics systems.

## Start Here

Recommended order:

1. Start with [beginner/01-csv-to-postgres/](beginner/01-csv-to-postgres/).
2. Build [beginner/04-retail-sales-pipeline/](beginner/04-retail-sales-pipeline/) when you know basic SQL and Python.
3. Move to [intermediate/02-retail-sales-warehouse/](intermediate/02-retail-sales-warehouse/) after learning modeling and warehousing.
4. Choose one advanced project only after you can explain why the architecture is useful.

# Setup

## Prerequisites

- Python
- `requests`
- `pandas`
- A public API or test API endpoint

## Steps

1. Choose an API endpoint that returns JSON.
2. Send a `GET` request with a timeout.
3. Check the response status code.
4. Parse the JSON response.
5. Convert records to a table.
6. Write a CSV file.
7. Log the output row count.

## Example Commands

```bash
python extract_api.py
```

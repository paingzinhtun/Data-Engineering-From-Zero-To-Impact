# Project 02: API To CSV

## Goal

Extract data from an API and save the response to a CSV file that can be used by later pipeline steps.

## What You Will Learn

- How APIs return JSON data
- How to use Python `requests`
- How to handle status codes and timeouts
- How to flatten simple JSON records
- How to write extracted data to CSV

## Business Scenario

A small ecommerce shop needs to extract order or product data from a platform API every day. This project starts with a simple API extractor before moving to databases and orchestration.

# Setup

## Prerequisites

- PostgreSQL
- Retail sample tables
- SQL client such as `psql`, DBeaver, or pgAdmin

## Steps

1. Load the practice retail database from `02-sql-and-databases/postgresql-practice/`.
2. Run beginner queries.
3. Modify queries for new questions.
4. Save useful queries.
5. Write a short business summary.

# Expected Output

By the end of this project, you should have:

- A PostgreSQL database
- A `raw_products` table
- Product rows loaded from CSV
- SQL validation queries
- A short README explaining the source data and load process

## Success Criteria

- Product row count matches the CSV.
- `product_id` is not null.
- `product_id` is unique.
- Prices are non-negative.
- The table can be queried from PostgreSQL.

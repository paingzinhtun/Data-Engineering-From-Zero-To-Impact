# Project 01: CSV To PostgreSQL

## Goal

Load a business CSV file into PostgreSQL and validate that the table contains the expected data.

## What You Will Learn

- How to inspect a CSV file
- How to design a simple table
- How to load rows into PostgreSQL
- How to run basic validation queries
- How to document a small data loading project

## Business Scenario

A small stationery shop tracks products in a spreadsheet. The business wants the product catalog loaded into PostgreSQL so sales and inventory analysis can use a reliable table instead of manual spreadsheet copies.

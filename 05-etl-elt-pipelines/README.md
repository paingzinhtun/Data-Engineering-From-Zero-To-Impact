# 05 - ETL And ELT Pipelines

## What This Section Covers

This section covers extraction, loading, transformation, batch processing, incremental loads, idempotency, retries, backfills, pipeline design, and operational documentation.

## Why It Matters In Data Engineering

Pipelines are the systems that move data from where it is created to where it can be used. Reliable pipelines reduce manual work, improve trust, and make analytics and automation repeatable.

## What You Should Be Able To Do

After studying this section, you should be able to:

- Explain ETL and ELT patterns.
- Design a simple batch pipeline.
- Handle full refresh and incremental loading.
- Make pipeline steps rerunnable.
- Plan for failures, retries, and backfills.
- Document pipeline inputs, outputs, and assumptions.
